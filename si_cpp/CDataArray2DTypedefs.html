Type Definitions for CDataArray2D

The CDataArray2D class is a template class that encapsulates the following 2D data arrays:

  • CDataArray2DFloat
  • CDataArray2DLong
  • CDataArray2DBool
  • CDataArray2DVector2f
  • CDataArray2DVector3f
  • CDataArray2DVector4f
  • CDataArray2DQuaternionf
  • CDataArray2DRotationf
  • CDataArray2DMatrix3f
  • CDataArray2DMatrix4f
  • CDataArray2DColor4f
  • CDataArray2DShape
