Type Definitions for CDataArray

The CDataArray class is a template class that encapsulates the following 1D data arrays:

  • CDataArrayFloat
  • CDataArrayLong
  • CDataArrayBool
  • CDataArrayVector2f
  • CDataArrayVector3f
  • CDataArrayVector4f
  • CDataArrayQuaternionf
  • CDataArrayRotationf
  • CDataArrayMatrix3f
  • CDataArrayMatrix4f
  • CDataArrayColor4f
  • CDataArrayShape
