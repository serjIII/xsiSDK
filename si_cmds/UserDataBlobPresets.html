UserDataBlob Presets

List of available UserDataBlob Property presets. These presets can be used to create UserDataBlob objects on X3DObjects, and partial or complete Clusters, Materials, Shaders, Clips, and Sources using any of the AddProperty methods (SceneItem.AddProperty, Material.AddProperty, Shader.AddProperty, Source.AddProperty, Clip.AddProperty).
