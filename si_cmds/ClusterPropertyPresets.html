ClusterProperty Property Presets

List of available ClusterProperty Property presets. These presets can be used to create ClusterProperty objects on partial or complete Clusters using the SceneItem.AddProperty method.

Warning Some of these presets can only be added to always-complete Clusters. Such cases are indicated in the table below. An always-complete cluster always covers all the components of the Geometry, even when new components are added by a modeling operation (see Cluster.IsAlwaysComplete for more information).
Note See Property Presets for property classification. Also see Alphabetical Listing of All Property Presets for a complete list of available property types.
