CreateMeshText

CreateMeshText

Description

Converts a text curve list object to a mesh.

Scripting Syntax

oReturn = CreateMeshText( PresetObj, [ImmediateMode] );

Return Value

Returns the 3D text object as an XSICollection object containing the Operator as its first member.

Parameters

Parameter Type Description
PresetObj String or a preset object (see SIGetPreset) One of the CurveListToMesh operator presets.

Default Value: If no preset is specified, a browser pops up

Possible Values:

Description:

CurveListToPlanarMeshForText CurveListToPlanarMeshForText operator preset
CurveListToSolidMeshForText CurveListToSolidMeshForText operator preset
ImmediateMode siOperationMode Determines whether to use immediate mode or persistent mode.

Default Value: siPersistentOperation

Examples

VBScript Example

' Create a planar mesh from a text curvelist
CreateMeshText "CurveListToPlanarMeshForText"
' Create a solid mesh from a text curvelist
CreateMeshText "CurveListToSolidMeshForText"
