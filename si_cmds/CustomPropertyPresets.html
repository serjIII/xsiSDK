CustomProperty Property Presets

List of available CustomProperty Property presets. These presets can be used to create CustomProperty objects on X3DObjects, partial or complete Clusters, Materials, Shaders, Clips, and Sources using any of the AddProperty methods (SceneItem.AddProperty, Material.AddProperty, Shader.AddProperty, Source.AddProperty, Clip.AddProperty).

Note See Property Presets for property classification. Also see Alphabetical Listing of All Property Presets for a complete list of available property types.
