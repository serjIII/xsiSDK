Primitive Presets

This chapter contains the list of available Primitive object presets organized by type. For information on the specific primitive types and a list of the primitive objects that match each type, click on the corresponding links below:

  • Alphabetical Listing of All Primitive Presets
Note Not all values are supported in every context; for example, the X3DObject.AddGeometry method only accepts some primitive presets with a specific geometry type, while the X3DObject.AddPrimitive method allows any preset in the alphabetical list of all primitives.
