Using the SDK Online Resource

Welcome to the Softimage|XSI Software Developer's Kit (SDK) online reference. The links in the top left pane are organized according to topic type, such as command, object, constant, or node. When you click on one of these topic links, a list of entry links appears in alphabetical order in the bottom left pane. When you can click on one of the entry links in the bottom left pane, the information for that entry appears in the main page on the right.
The topic types are organized as follows:

Native XSI Scripting Commands

You can look for commands either by category or by name (arranged alphabetically). Click on Commands to see topic links for either of these options.

The Object Model

Objects, Methods and Properties all have their own links. You can also use the Collections and Constants topic links to see a list of entry links related to these special types of entry. The Object Hierarchy contains a view of how the various object interfaces inherit methods and properties.

ICE Node Reference

You can look for ICE nodes either by category or by name (arranged alphabetically). Click on ICE Nodes to see topic links for either of these options.
