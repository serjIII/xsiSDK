CreateLayoutGroupInICECompoundNode

CreateLayoutGroupInICECompoundNode

Introduced

v13.0 (2015)

Description

Create a layout group and add the specified exposed ports and layout groups into it. If the specified ports and layout groups are part of different layout groups, the new layout group will be created underneath a common parent layout group.

Scripting Syntax

oReturn = CreateLayoutGroupInICECompoundNode( Compound, Names, [Group] );

Return Value

Returns the name of the group. It may be different from the specified name, in order to make unique.

Parameters

Parameter Type Description
Compound Object Name (or a pointer to it). ICECompoundNode which you want to edit.
Names Array of Strings Names of the ports and layout groups to put in the new layout group.
Group String Name of the group.

Default Value: "Group"

Examples

Python Example

#
# This example shows how to move ICE compound ports and the layout groups.
#
from sipyutils import si		# win32com.client.Dispatch('XSI.Application')
si = si()
si.NewScene( "", False )
si.CreatePrim("Grid", "MeshSurface", "", "")
si.ICEFlow_CreateParticles("", "", "")
# Create a new layout group and move the existing layout group and ports into it.
si.CreateLayoutGroupInICECompoundNode("pointcloud.pointcloud.ICETree.Simulation_Root", ["Primary State", "Trigger1", "State_Color"])
