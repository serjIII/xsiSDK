Shader Presets

List of available Shader presets which can be used with the following commands and methods:

Commands Methods
  • AddPresetTextureLayer
  • ApplyShader
  • BlendInTextureLayers
  • BlendInTextureLayersInsp
  • BlendInTextureLayersWithPorts
  • BlendInTextureLayersWithPortsInsp
  • CreateShaderFromPreset
  • InsertShader
  • ReplaceShaderWithPreset
  • SIAddShaderEntryToOverride
  • SIApplyShader
  • SIApplyShaderToCnxPoint
  • SICreateMaterial
  • SceneItem.AddMaterial
  • Parameter.Connect
  • Parameter.ConnectFromFile
  • Parameter.ConnectFromPreset
  • XSIFactory.CreateObjectFromPreset
Warning Although most of these presets will work with the above commands and methods, there are some restrictions based on the target object and the type of shader to be applied. For example, the AddMaterial method can only use presets that connect to a material or shader, so conversion and math shaders will fail. In that case, you should use one of the Connect methods on the Parameter object or the SIApplyShaderToCnxPoint command to specify the precise way to connect.

In addition, a number of shaders can be used as global shaders when they are applied to Pass objects via the SIApplyShaderToCnxPoint command. These include the Environment, Output, and Volume shader presets.

The presets are organized by category:

  • Environment
  • Geometry
  • Lens
  • Light
  • Lightmap
  • Output
  • Material
  • Texture
  • Volume
  • Comment
  • RealTime
