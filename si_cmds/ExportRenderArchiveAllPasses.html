ExportRenderArchiveAllPasses

ExportRenderArchiveAllPasses

Introduced

v6.0

Description

Exports all passes in the scene to render archives. The render archives can subsequently then be used by a standalone version of the rendering engine that exported it.

Scripting Syntax

ExportRenderArchiveAllPasses( [Passes] );

Parameters

Parameter Type Description
Passes String Passes to render

Default Value: all passes

Examples

VBScript Example

'
'	This example creates a cone with some animation on it and then exports
'	all passes to an archive.
'
' Create a new scene 
NewScene , false
' Add a cone with some animation
Set oCone = CreatePrim( "Cone", "MeshSurface" )
SaveKey oCone & ".kine.local.posx," _
	& oCone & ".kine.local.posy," _
	& oCone & ".kine.local.posz", _
	1
SetValue "PlayControl.Key", 100
SetValue "PlayControl.Current", 100
Translate oCone, _
	6.94824213992454, _
	2.46082959410157, _
	-0.246082959410157, _
	siRelative, _
	siView, _
	siObj, _
	siXYZ
SaveKey oCone & ".kine.local.posx," _
	& oCone & ".kine.local.posy," _
	& oCone & ".kine.local.posz", _
	5
' Export all passes as render archives
ExportRenderArchiveAllPasses
