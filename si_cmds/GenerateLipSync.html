GenerateLipSync

GenerateLipSync

Introduced

v12.0 (2014)

Description

Generate lip-sync animation for an Evolver character

Scripting Syntax

oReturn = GenerateLipSync( [Model], AudioFileName, [TextFileName], [Language], [Resolutions], [MappingProfilePath] );

Return Value

Returns the lip-sync action clip.

Parameters

Parameter Type Description
Model String Evolver Model

Default Value: Current selection

AudioFileName String Audio file from which the lip-sync animation is generated
TextFileName String Text transcript of the speech
Language String Language of the speech

Default Value: en

Possible Values:

Description:

en English
jp Japanese
Resolutions String List of mesh resolutions on which the lip-sync animation is applied. The following abbreviations should be used to indicate the resolutions.

Default Value: hmlc

Possible Values:

Description:

h High
m Medium
l Low
c Crowd
MappingProfilePath String Path to the phoneme-to-viseme mapping profile
