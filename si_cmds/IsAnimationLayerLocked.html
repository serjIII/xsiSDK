IsAnimationLayerLocked

IsAnimationLayerLocked

Introduced

v13.0 (2015)

Description

Returns True if the specified animation layer is locked, False otherwise.

Note: A value of 0 for the layer will be ignored because the base animation layer cannot be locked.

Scripting Syntax

oBoolean = IsAnimationLayerLocked( [InputObjs], [Layer] );

Return Value

Boolean

Returns True if the specified animation layer is locked, False otherwise.

Parameters

Parameter Type Description
InputObjs String List of objects.

Default Value: Currently selected objects

Layer Long Animation layer to test. Only layers greater than 0 can be tested.

Default Value: Current animation layer (-1)

Examples

VBScript Example

'
' This example shows how to test if an animation is locked
' and how to unlock and lock it, if it was locked
'
' Make the base animation layer the current
SetCurrentAnimationLayer null, 0
' Create a cube and animate its posx parameter
CreatePrim "Cube", "NurbsSurface", "MyCube"
AddFCurve "MyCube.kine.local.posx"
SaveKey "MyCube.kine.local.posx", 1, -2 
SaveKey "MyCube.kine.local.posx", 30, 5 
' Make sure we have an animation layer (in addition of the base layer)
if GetNumberOfAnimationLayers( null ) = 1 then 
	AddAnimationLayer
end if
' Add a key on the animation layer for the posx FCurve of cube.
' If the animation layer is locked, unlock it, add the key,
' and lock it back
SetCurrentAnimationLayer null, 1
animLayerWasLocked = false
if IsAnimationLayerLocked( null, 1) then
	animLayerWasLocked = true
	UnlockAnimationLayer null, 1
end if
SaveKey "MyCube.kine.local.posx", 1, , , , False
if animLayerWasLocked then
	LockAnimationLayer null, 1
end if
