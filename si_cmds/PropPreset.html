Property Presets

This chapter contains the list of available Property object presets organized by class type (that is, the class of object returned by one of the AddProperty methods). These presets can be used with the following methods:

  • SceneItem.AddProperty
  • Material.AddProperty
  • Shader.AddProperty
  • Source.AddProperty
  • Clip.AddProperty
Warning Not all values are supported in every context; for example, some CustomProperty Presets such as Vertex Color can only be applied to a complete Cluster with the SceneItem.AddProperty method, while some CustomProperty presets such as Annotation can be applied to X3DObjects, partial or complete Clusters, Materials, Shaders, Sources, or Clips.
