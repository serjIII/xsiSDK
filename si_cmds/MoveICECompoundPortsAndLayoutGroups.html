MoveICECompoundPortsAndLayoutGroups

MoveICECompoundPortsAndLayoutGroups

Introduced

v13.0 (2015)

Description

Moves the ICE compound ports and its PPG layout group hierarchy. You can pass a list containing the name of compound ports or the name of PPG layout groups.

Note: It's able to move a port within a same group, or into another group. Also can move a group that will lead of moving a bunch of ports inside it.

Scripting Syntax

MoveICECompoundPortsAndLayoutGroups( Compound, Names, Marker, [MoveAction] );

Parameters

Parameter Type Description
Compound Object Name (or a pointer to it). ICECompoundNode which you want to edit.
Names Array of Strings Names of the ports and PPG Layout groups to move.
Marker String Specifies the marker item(port or layout group) as the moving destination.
MoveAction Integer Specifies where item will be moved: first child, before/after the reference item, or last child.

Default Value: 1

Possible Values:

Description:

0 Move to become the first child.
1 Move just before the marker item.
2 Move just after the marker item.
3 Move to become the last child.

Examples

Python Example

#
# This example shows how to move ICE compound ports and the layout groups.
#
from sipyutils import si		# win32com.client.Dispatch('XSI.Application')
si = si()
si.NewScene( "", False )
si.CreatePrim("Grid", "MeshSurface", "", "")
si.ICEFlow_CreateParticles("", "", "")
# Move the ports to "General Values" layout group that's nested underneath "Initial Values" layout group
aProps = si.GetICECompoundPortProperties("pointcloud.pointcloud.ICETree.Emit_from_Geometry.Color") 
si.EditExposedParamInICECompoundNode("pointcloud.pointcloud.ICETree.Emit_from_Geometry.Color", aProps[0], aProps[1], aProps[2], aProps[3], aProps[4], aProps[5], aProps[6], "Initial Values;General Values" )
aProps = si.GetICECompoundPortProperties("pointcloud.pointcloud.ICETree.Emit_from_Geometry.Shape") 
si.EditExposedParamInICECompoundNode("pointcloud.pointcloud.ICETree.Emit_from_Geometry.Shape", aProps[0], aProps[1], aProps[2], aProps[3], aProps[4], aProps[5], aProps[6], "Initial Values;General Values" )
# Move the port to become the first child
si.MoveICECompoundPortsAndLayoutGroups("pointcloud.pointcloud.ICETree.Emit_from_Geometry", ["State_ID"], "General Values", 0)
# Move the port to become the last child
si.MoveICECompoundPortsAndLayoutGroups("pointcloud.pointcloud.ICETree.Emit_from_Geometry", ["Mass"], "General Values", 3)
# Move the port "Mass" before "Color"
si.MoveICECompoundPortsAndLayoutGroups("pointcloud.pointcloud.ICETree.Emit_from_Geometry", ["Shape", "Mass"], "Color", "")
# Move the layout group "General Values" after "Initial Values"
si.MoveICECompoundPortsAndLayoutGroups("pointcloud.pointcloud.ICETree.Emit_from_Geometry", ["General Values"], "Initial Values", 2)
