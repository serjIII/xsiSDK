View Attributes

Some view types have attributes that you can set and/or get. For example, you can control the target content of an Object view, set the default URL for a Net View, or specify a file to open in a text editor. Note Most attribute values are represented as strings unless otherwise indicated, no matter what value they may seem like. For example, boolean true and false must be specified as the strings "true" and "false" and any numeric values, such as the index number of the tab to activate, must be specified as "2", "7", "13", "20", etc.
