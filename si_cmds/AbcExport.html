AbcExport

Introduced

v13.0 (2015)

Description

Exports objects into Abc.

Scripting Syntax

oReturn = AbcExport( FilePathName, [StartFrame], [EndFrame], [InputObjs], [IncludeParentHierarchy], [ArchiveFormat], [AttributesPointcloud], [AttributesPolymesh], [AttributesOtherPrim] );

Return Value

True if successful, False otherwise.

Parameters

Parameter Type Description
FilePathName String Full path and filename of Abc file to export.
StartFrame Long Start frame from which the animation is exported.

Default Value: 1

EndFrame Long End time of the animation.

Default Value: 100

InputObjs List Objects to be exported.

Default Value: Current selection. If nothing is selected, the entire scene will be exported.

IncludeParentHierarchy Boolean True if the parent hiearchy should be included. If the objects to be exported are nested under parents with non-identity transform, set this flag to true to preserve their global transforms.

Default Value: False

ArchiveFormat String The format of the Abc archive.

Default Value: Ogawa

Possible Values:

Description:

HDF5
Ogawa
AttributesPointcloud String String to specify Pointcloud attributes to be exported.
AttributesPolymesh String String to specify Polymesh attributes to be exported.
AttributesOtherPrim String String to specify attributes to be exported for other primitives.
