siViewMode

siViewMode

Description

Viewport display modes used by the CGraphicSequencer class for the Graphic Sequencer.

C# Syntax

siViewMode.siAll                                                            // 0
siViewMode.siBoundingBoxMode                                                // 12
siViewMode.siWireframe                                                      // 13
siViewMode.siDepthCue                                                       // 14
siViewMode.siHiddenLineRemoval                                              // 15
siViewMode.siConstant                                                       // 16
siViewMode.siShaded                                                         // 17
siViewMode.siTextured                                                       // 18
siViewMode.siTexturedDecal                                                  // 19
siViewMode.siCustom                                                         // 20
siViewMode.siRealtimePortMaterial                                           // 21

Constant Value Description
siAll 0 All modes.
siBoundingBoxMode 12 Bounding box.
siWireframe 13 Wireframe.
siDepthCue 14 Depth Cue.
siHiddenLineRemoval 15 Hidden Line Removal.
siConstant 16 Constant.
siShaded 17 Shaded.
siTextured 18 Textured.
siTexturedDecal 19 Textured Decal.
siCustom 20 Custom view.
siRealtimePortMaterial 21 Realtime.

Applies To

CGraphicSequencer::RegisterViewportCallback CGraphicSequencer::RegisterDisplayCallback CGraphicSequencer::RenderSceneUsingMode CGraphicSequencer::RenderListUsingMode
