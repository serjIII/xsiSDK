Geometry2D.Neighborhood1D

Geometry2D.Neighborhood1D

Description

Returns an Array of indices (Integers) of the 1D neighborhood of a given 2D geometry within a given distance.

Parameters

Parameter Type Description
compIdx Integer Index of the component

Possible Values:

Description:

0 <= compidx="">< number="" of="" 2d="" component="" /> The compIdx must be valid
distance Integer Distance to take into account

Examples

VBScript Example

'Create a cube
CreatePrim "Cube", "MeshSurface"
set oSelList = GetValue("SelectionList")
set oItem = oSelList(0)
set oGeometry = oItem.obj
oNb2D = oGeometry.Nb2D
set o2DComponent = oGeometry.Geometry2D
' Loop on all the 2D Component
for i = 0 to oNb2D - 1
o1DNeighborhood = o2DComponent.Neighborhood1D( i, 1 )
for j = LBound( o1DNeighborhood, 1 ) to UBound( o1DNeighborhood, 1 )
LogMessage "Component [" & i & "] 1D neighbor : " & o1DNeighborhood(j) 
next
next
