LinktabRuleCollection

LinktabRuleCollection

Description

A collection of VARIANTS that represent a subset of path translation that linktab uses. You can get this object by using Linktab.Rules. See Linktab object documentation for more information about linktab.

Examples

VBScript Example

'VBScript example
' Get a LinktabRuleCollection from the Linktab object
	set l_linktab = xsiutils.Linktab
	if l_linktab.Filename(szFileName) = "True" then
		set l_rulesColl =	l_linktab.Rules
		' Display each rule in the collection
		For each l_line in l_rulesColl 
			LogMessage l_line 
		Next 
		' Display the rules count
		LogMessage l_rulesColl.Count
		' Display the rules using Item property as the default property
		For i=0 to l_rulesColl.Count-1
			LogMessage  l_rulesColl(i)
		Next
		' Display the rules using Item property as the default property
		For i=0 to l_rulesColl.count-1
			LogMessage  l_rulesColl.Item(i)
		Next
		' Display a subset of rules
		set l_SubRules = l_rulesColl.Filter(siRuleKind_NT2Unix)
		' Display the sub (if any)
		If l_SubRules.Count > 0 Then
			For i=0 to l_SubRules.Count-1
				LogMessage  l_SubRules(i)
			Next
		End If
	else
		LogMessage "Linktab file not set"
	End if
