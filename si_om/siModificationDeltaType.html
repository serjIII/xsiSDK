siModificationDeltaType

siModificationDeltaType

Introduced

v6.0

Description

Type of delta modification on a referenced model.

C# Syntax

siModificationDeltaType.siModificationDeltaActiveResolution                 // 0
siModificationDeltaType.siModificationDeltaRemoveAnimation                  // 1
siModificationDeltaType.siModificationDeltaConstraint                       // 2
siModificationDeltaType.siModificationDeltaUpVectorConstraint               // 3
siModificationDeltaType.siModificationDeltaOperator                         // 4
siModificationDeltaType.siModificationDeltaExpression                       // 5
siModificationDeltaType.siModificationDeltaStaticValue                      // 6
siModificationDeltaType.siModificationDeltaFCurve                           // 7
siModificationDeltaType.siModificationDeltaGroupRelation                    // 8
siModificationDeltaType.siModificationDeltaCluster                          // 9
siModificationDeltaType.siModificationDeltaClusterProperty                  // 10
siModificationDeltaType.siModificationDeltaProperty                         // 11

Constant Value Description
siModificationDeltaActiveResolution 0 Store the active resolution of a sub-reference model.
siModificationDeltaRemoveAnimation 1 Store animation removals on the current reference model.
siModificationDeltaConstraint 2 Store Constraints.
siModificationDeltaUpVectorConstraint 3 Store up vector constraints.
siModificationDeltaOperator 4 Store Operators.
siModificationDeltaExpression 5 Store Expressions.
siModificationDeltaStaticValue 6 Store static values.
siModificationDeltaFCurve 7 Store FCurves.
siModificationDeltaGroupRelation 8 Store group relations (Objects added to a group, a layer or a pass).
siModificationDeltaCluster 9 Clusters on Geometry.
siModificationDeltaClusterProperty 10 Cluster Properties on Geometry.
siModificationDeltaProperty 11 Added Properties.

Applies To

Delta.AddAction Delta.RemoveAction SIObject.Type
