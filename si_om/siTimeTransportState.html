siTimeTransportState

siTimeTransportState

Introduced

v13.0 (2015)

Description

Describes the current state of the transport. You are pass a value in the TimeTransport callback which is a combination of these flags.

C# Syntax

siTimeTransportState.siTTStateStop                                          // 1
siTimeTransportState.siTTStatePlay                                          // 4
siTimeTransportState.siTTStateSeek                                          // 8192
siTimeTransportState.siTTStateTimeChange                                    // 16384
siTimeTransportState.siTTStateContextChange                                 // 32768
siTimeTransportState.siTTStateScrubChange                                   // 65536
siTimeTransportState.siTTStateSeekChange                                    // 131072
siTimeTransportState.siTTStateDeviceChange                                  // 262144
siTimeTransportState.siTTStateFrameDropChange                               // 524288

Constant Value Description
siTTStateStop 0x00001 Stop
siTTStatePlay 0x00004 Play
siTTStateSeek 0x02000 Seek
siTTStateScrub 0x10000 Scrub
siTTStateTimeChange 0x04000 Time change
siTTStateContextChange 0x08000 Context change
siTTStateScrubChange 0x10000 Scrub change
siTTStateSeekChange 0x20000 Seek change
siTTStateDeviceChange 0x40000 Device change
siTTStateFrameDropChange 0x80000 Frame drop change

Applies To

TimeTransport Context Attributes
