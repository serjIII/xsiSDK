siShadingModel

siShadingModel

Description

Defines the material shading model

C# Syntax

siShadingModel.siUndefinedShadingModel                                      // -1
siShadingModel.siShadingModelConstant                                       // 0
siShadingModel.siShadingModelLambert                                        // 1
siShadingModel.siShadingModelBlinn                                          // 2
siShadingModel.siShadingModelPhong                                          // 3
siShadingModel.siShadingModelShadow                                         // 4
siShadingModel.siShadingModelColorVertex                                    // 5

Constant Value Description
siUndefinedShadingModel -1 Undefined shading model
siShadingModelConstant 0 Constant shading
siShadingModelLambert 1 Lambert shading
siShadingModelBlinn 2 Blinn shading
siShadingModelPhong 3 Phong shading
siShadingModelShadow 4 Shadow shading
siShadingModelColorVertex 5 Color vertex shading

Applies To

OGLMaterial.ShadingModel OGLMaterial::GetShadingModel
