PluginRegistrar.HelpButton

PluginRegistrar.HelpButton

Introduced

v13.0 (2015)

Description

Sets or returns, as a String, the help button file replacing the factory "?" button in the property pages of the plug-in items of this plug-in being registered. An empty string is returned if the plug-in has no help button file.

The help button will be shown in a plug-in item property page only if there is a help file associated with the plug-in item.

The plug-in help button file must be a BMP file with a height of 18 pixels and a width no wider than 100 pixels. If the help button file doesn't respect the criteria, the factory "?" button will be displayed in the plug-in items property pages.

By default, Softimage looks for a plug-in help button file in the "doc" subfolder of the plug-in installation folder. Softimage assumes the help button file has the same name as the plug-in, with the "_HelpButton" string appended to it. For example, if the plug-in MyPlugin.js is located in the folder "C:\MyWorkgroup\Application\Plugins" then Softimage looks in the folder "C:\MyWorkgroup\Application\Plugins\Doc" for a help button file named MyPlugin_HelpButton.bmp.

If a plug-in help button file has a different name or location, you can set this property to specify the name of help button file and, optionally, its location.

C# Syntax

// get accessor
String rtn = PluginRegistrar.HelpButton;
// set accessor
PluginRegistrar.HelpButton = String;

Examples

1. VBScript Example

'
' This example shows how to specify help button file to a plug-in 
' and to specific plug-in items. The help button files are assumed
' to be in the "doc" subfolder of the plug-in installation folder.
'
Function XSILoadPlugin( in_reg )
	' register plugin information
	in_reg.Author = "Autodesk Softimage" 
	in_reg.Name = "Testing Help Button"
	in_reg.Major = 1
	in_reg.Minor = 0 
	in_reg.Help = "Testing_Help_Button.htm"
	in_reg.HelpButton = "Testing_Help_Button.bmp"
	' no need to set the help button file for this property,
	' the "MyFirstProperty_HelpButton.bmp" file located in the
	' "doc" subfolder will automatically be found by Softimage
	set item = in_reg.RegisterProperty( "MyFirstProperty" )
	'item.HelpButton = "MyFirstProperty_HelpButton.bmp"
	' this help button file is in the same folder as the plug-in
	set item = in_reg.RegisterProperty( "MySecondProperty" )
	sHelpButton = "MyHelpButtonFile.bmp"
	item.HelpButton = XSIUtils.BuildPath( in_reg.OriginPath, sHelpButton )
	' no need to set the help button file for this plug-in item,
	' the help button file set to the plug-in is suitable for this plug-in item
	' (considering there is no "MyThirdProperty_HelpButton.bmp" file located
	' in the "doc" subfolder of the plug-in installation folder)
	in_reg.RegisterProperty "MyThirdProperty"
	XSILoadPlugin = true
end Function

2. JScript Example

/*
	This example shows how to get the plug-in items help button files
*/
var items = Application.Plugins( "Testing Help Button" ).Items;
for( var i = 0; i < items.Count; i++ ) 
{
	Application.LogMessage( items( i ).Name + " help button file is " + items( i ).HelpButton );
}
