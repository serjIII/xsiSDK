TimeControl.ClipIn

TimeControl.ClipIn

Description

Returns the first frame of the source that is used by the clip as a Double. Increase this value to trim a clip so that it begins part way through the duration of the source.

On the Time Control property editor, this appears on the General tab under Source Clipping.

C# Syntax

// get accessor
Double rtn = TimeControl.ClipIn;
