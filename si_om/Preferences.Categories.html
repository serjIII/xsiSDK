Preferences.Categories

Preferences.Categories

Description

Returns a ProjectItemCollection containing one ProjectItem for each category nested under the Preferences node.

C# Syntax

// get accessor
Object Preferences.get_Categories();
// set accessor
Preferences.set_Categories( String pArrayVal );

Examples

VBScript Example

set oCategories = Application.Preferences.Categories
For each oCat in oCategories
Application.LogMessage "The Category " & oCat & " has " & oCat.Parameters.Count & " preferences item(s)"
