siBranchFlag

siBranchFlag

Description

The branch flag is used to determine whether we are referring to the object as a node or as a branch.

C# Syntax

siBranchFlag.siNode                                                         // 0
siBranchFlag.siBranch                                                       // 1
siBranchFlag.siUnspecified                                                  // 3
siBranchFlag.siNotSelected                                                  // 4

Constant Value Description
siNode 0 Node
siBranch 1 Branch
siUnspecified 3 Unspecified
siNotSelected 4 Not selected
