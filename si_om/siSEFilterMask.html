siSEFilterMask

siSEFilterMask

Introduced

v5.0

Description

The string constants to use to set the FilterMask on any scene explorer using the View.SetAttributeValue "FilterMask" property.

C# Syntax

StringModule.siSEFilterActionNodes                                          // Action
StringModule.siSEFilterAllNodeTypes                                         // AllNodeTypes
StringModule.siSEFilterAnimatableNodes                                      // Animatable

StringModule.siSEFilterClusterNodes                                         // Cluster
StringModule.siSEFilterGroupNodes                                           // Group
StringModule.siSEFilterKeywordAll                                           // AllKeywords
StringModule.siSEFilterKeywordAnimation                                     // Animation
StringModule.siSEFilterKeywordCustom                                        // Custom
StringModule.siSEFilterKeywordGeneral                                       // General
StringModule.siSEFilterKeywordKinematics                                    // Kinematics
StringModule.siSEFilterKeywordModeling                                      // Modeling
StringModule.siSEFilterKeywordRendering                                     // Rendering
StringModule.siSEFilterKeywordViewing                                       // Viewing

StringModule.siSEFilterMiscellaneousNodes                                   // Miscellaneous
StringModule.siSEFilterModelNodes                                           // Model

Constant Value Description
siSEFilterActionNodes Action Show only Action nodes.
siSEFilterAllNodeTypes AllNodeTypes Show All Node types.
siSEFilterAnimatableNodes Animatable Show only Animatable parameter nodes.
siSEFilterAnimatedNodes Animated Show only Animated parameter nodes.
siSEFilterClusterNodes Cluster Show only Cluster nodes.
siSEFilterGroupNodes Group Show only Group nodes.
siSEFilterKeywordAll AllKeywords Show nodes with any keyword set.
siSEFilterKeywordAnimation Animation Show nodes with Animation keyword set.
siSEFilterKeywordCustom Custom Show nodes with Custom keyword set.
siSEFilterKeywordGeneral General Show nodes with General keyword set.
siSEFilterKeywordKinematics Kinematics Show nodes with Kinematics keyword set.
siSEFilterKeywordModeling Modeling Show nodes with Modeling keyword set.
siSEFilterKeywordRendering Rendering Show nodes with Rendering keyword set.
siSEFilterKeywordViewing Viewing Show nodes with Viewing keyword set.
siSEFilterMaterialNodes Material Show only Material nodes.
siSEFilterMiscellaneousNodes Miscellaneous Show only Miscellaneous nodes.
siSEFilterModelNodes Model Show only Model nodes.
siSEFilterObjectNodes Object Show only Object nodes.
siSEFilterOperatorNodes Operator Show only Operator nodes.
siSEFilterParameterNodes Parameter Show only regular Parameter (non-animatable) nodes.
siSEFilterPrimitiveNodes Primitive Show only Primitive nodes.
siSEFilterPropertyNodes Property Show only Property nodes.
siSEFilterSceneNodes Scene Show only Scene nodes.
siSEFilterShaderNodes Shader Show only Shader nodes.
