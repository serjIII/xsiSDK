XSIFactory.CreateObjectFromPreset2

XSIFactory.CreateObjectFromPreset2

Introduced

v11.0 (2013)

Description

Creates an object from a preset and optional preset family name. This method is similar to XSIFactory.CreateObjectFromPreset but specific to the Python language.

C# Syntax

Object XSIFactory.CreateObjectFromPreset2( String Name, String family );

Scripting Syntax

oReturn = XSIFactory.CreateObjectFromPreset2( PresetName, [Family] );

Return Value

SIObject

Parameters

Parameter Type Description
PresetName String Name of the preset. Can be any value from the following preset lists: Constraint Presets, Operator Presets, Model Presets, Pass Presets, Primitive Presets, Property Presets, or Shader Presets

Default Value: ""

Family String The name of the object's family; for example, "Operator"

Default Value: If not specified then the first preset found matching the PresetName is used.
