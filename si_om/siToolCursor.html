siToolCursor

siToolCursor

Introduced

v2012(10)

Description

ToolContext cursors.

C# Syntax

siToolCursor.siArrowCursor                                                  // 0
siToolCursor.siCrossCursor                                                  // 1
siToolCursor.siHandCursor                                                   // 2
siToolCursor.siPenCursor                                                    // 3
siToolCursor.siDropperCursor                                                // 4
siToolCursor.siSelectCursor                                                 // 5
siToolCursor.siTranslateCursor                                              // 6
siToolCursor.siRotateCursor                                                 // 7
siToolCursor.siScaleCursor                                                  // 8
siToolCursor.siZoomCursor                                                   // 9
siToolCursor.siDragXYCursor                                                 // 10
siToolCursor.siDragXCursor                                                  // 11
siToolCursor.siDragYCursor                                                  // 12
siToolCursor.siDotCursor                                                    // 13

Constant Value Description
siArrowCursor 0 Standard arrow cursor.
siCrossCursor 1 Crosshair cursor.
siHandCursor 2 Pointing hand cursor.
siPenCursor 3 Pen cursor.
siDropperCursor 4 Eye dropper cursor.
siSelectCursor 5 Selection cursor.
siTranslateCursor 6 Translate cursor.
siRotateCursor 7 Rotate cursor.
siScaleCursor 8 Scale cursor.
siZoomCursor 9 Magnifying glass cursor.
siDragXYCursor 10 4-way arrow cursor.
siDragXCursor 11 Horizontal arrow cursor.
siDragYCursor 12 Vertical arrow cursor.
siDotCursor 0 Small dot cursor.

Applies To

ToolContext
