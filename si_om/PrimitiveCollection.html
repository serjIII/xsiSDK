PrimitiveCollection

Introduced

v1.5

Description

A collection of Primitive objects. This collection is 0-based.

Methods

Filter Find GetAsText  
       

Properties

Count Item    
       

Examples

VBScript Example

'VBScript example
set oRoot = ActiveProject.ActiveScene.Root
set oPrims = oRoot.Primitives
LogMessage typename(oPrims)
LogMessage "Number of primitives : " & oPrims.Count
for each oPrim in oPrims
	LogMessage oPrim.Name
next
