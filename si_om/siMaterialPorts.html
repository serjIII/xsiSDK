Description

Port choice for the SIGetMaterialShader command

C# Syntax

siMaterialPorts.siMaterialPortSurface                                       // 1
siMaterialPorts.siMaterialPortVolume                                        // 2
siMaterialPorts.siMaterialPortEnvironment                                   // 3
siMaterialPorts.siMaterialPortContour                                       // 4
siMaterialPorts.siMaterialPortDisplacement                                  // 5
siMaterialPorts.siMaterialPortShadow                                        // 6
siMaterialPorts.siMaterialPortPhoton                                        // 7
siMaterialPorts.siMaterialPortPhotonVol                                     // 8
siMaterialPorts.siMaterialPortBumpMap                                       // 9

Constant Value Description
siMaterialPortSurface 1 Surface shader port
siMaterialPortVolume 2 Volume shader port
siMaterialPortEnvironment 3 Environment shader port
siMaterialPortContour 4 Contour shader port
siMaterialPortDisplacement 5 Displacement shader port
siMaterialPortShadow 6 Shadow shader port
siMaterialPortPhoton 7 Photon shader port
siMaterialPortPhotonVol 8 Photon volume shader port
siMaterialPortBumpMap 9 Bump map shader port

Applies To

SIGetMaterialShader
