Getting Started with the Plug-in Wizard

The best way to get started with developing a self-installing plug-in is to use one of the Softimage SDK wizards to generate the initial code.

For C++ and C#, the wizards also generate a Visual Studio 2010 or Visual Studio 2012 project (.vcxproj), a Visual C++ project (.cpp), and a GNU makefile.

There are specific wizards for operators, commands, properties, and others.

To generate the code for an empty self-installing plug-in

  1. Select File Plug-in Manager. The Plug-in Manager dialog appears.

    NOTE:In the Tools Development Environment layout, you can click the Plug-ins tab to switch to the Plug-in Manager view.

  2. Select File New Empty Plug-in.

    The Plug-in Wizard appears.

  3. In the Plug-in Name box, type a name for the plug-in.

    The first character in a plug-in name must be a letter. You can use letters, numbers, underscore (_) characters, or spaces for the subsequent characters.

    If you include spaces in the plug-in name, the wizard replaces the spaces with underscores when it creates the plug-in file.

  4. In the Author box, type a name.

  5. In the Language drop-down list, select the language.

  6. In the Output Directory box, select or type the location to create the plug-in.

  7. Click OK to generate the code.

    If you selected a scripting language, the wizard also loads the plug-in.
