Providing Help Files

You can provide an HTML page or a web address (URL) as a part of your package. Net View loads the page when the help icon (?) is clicked.

To specify a help location and a help file icon for an on-the-fly custom property:

  • Use the PPGLayout.SetAttribute or PPGLayout::PutAttribute method with the siUIHelpFile enum. For example: 

    oLayout.SetAttribute(siUIHelpFile, "http://www.softimage.com/support/");
  • Use the siUIHelpButtonFile attribute to specify a help icon file for the property page. The help icon file replaces the default ? icon in the property page. The help icon must be a bitmap file (*.bmp). It must have a height of 18 pixels and a width not exceeding 100 pixels. 
    oLayout.SetAttribute(siUIHelpButtonFile, " MyHelpButtonFile.bmp");

To specify a help location and a help icon file for a plug-in based (self-installing) custom property:

Do the following:

  • Use the SIObject.Help or PluginItem::PutHelp or PluginRegistrar::PutHelp property in the XSILoadPlugin callback: 

    in_reg.Help = "http://www.softimage.com/support/";

  • Name your HTML file with the name of your plug-in and add it to the doc subfolder. Softimage finds your HTML page automatically. For example, following is a file structure for a typical self-installing plug-in containing a custom property: 

    C:\Public\Plugins\MyFabulousProperty.dll
C:\Public\Plugins\doc\MyFabulousProperty.html
  • Use PluginItem.PutHelpButton or PluginItemRegistrar.PutHelpButton to specify a help icon file. The help icon must be a bitmap file (*.bmp). It must have a height of 18 pixels and a width not exceeding 100 pixels.

  • If the self-installing property is in an Add-on directory, use the add-on help mechanism. See Add Help for the Add-on.
