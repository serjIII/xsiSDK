Moving Around in the Timeline

There are a series of commands that move to frames or keys. These are the scripting equivalent of the playback control buttons for moving to the next frame or key. These are:

  • FirstFrame

  • PrevFrame

  • NextFrame

  • LastFrame

These commands are basically shortcuts for changing the PlayControl.Current parameter value to the target frame number. For example, assuming you are on frame 18 and you want to go to the next frame. You can either use the NextFrame command or set the PlayControl.Current parameter to 9.

There is another series of commands that do not represent a simple parameter change. These navigate through the timeline based on the fcurve keys set:

  • FirstKey

  • PrevKey

  • NextKey

  • LastKey
