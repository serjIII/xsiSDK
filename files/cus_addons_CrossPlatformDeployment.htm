Cross-Platform Deployment

You can create workgroups and add-on packages that support both Windows and Linux.

Shader and Self-installing Plug-in Deployment

Create separate subfolders (such as linux\x64) that map to each platform. For plug-ins, this is the folder structure: 

\Application
	\Plugins
		\bin
			\nt-x86-64\MyPlugin.dll
			\linux\x64\MyPlugin.so

Softimage scans the folders according to the current machine architecture. Use the same basename for each plug-in library file.

NOTE:When Softimage starts, the platform-specific directories are searched, even if the Scan sub-folders preference is turned off.
