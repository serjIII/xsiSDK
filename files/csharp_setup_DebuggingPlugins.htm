Debugging Plug-ins

To debug the plug-in:

  1. Compile a Debug version of your plug-in and then start Softimage.

  2. In Visual Studio, select Debug Attach to Process.

  3. In the Attach to Processes dialog, select XSI.exe in the list of Available Processes.

  4. Click Attach.

  5. Set some breakpoints and start debugging.
