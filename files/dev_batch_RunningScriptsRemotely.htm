Running Scripts Remotely

You can run Autodesk Softimage in batch mode remotely using Telnet or a remote shell (rsh):

  • On Linux, use xsi -script or xsi -R as usual.

  • On Windows, use xsibatch -script or xsibatch -R instead. The xsibatch command supports the same options as xsi. However, the beta version of the Telnet daemon included in some versions of the Windows Resource Kit (TELNETD) is not supported.
