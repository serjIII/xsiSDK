Setting Up the Developer Environment

Custom displays use Softimage's C++ API. Before you begin, make sure you have set up your environment as described in Building C++ Plug-ins.

In addition to these settings, the following header files are required when developing custom displays. Their classes and functions are briefly described in the following section. For detailed information, see the C++ API Reference.

  • xsi_viewcontext.h

  • xsi_viewnotification.h
