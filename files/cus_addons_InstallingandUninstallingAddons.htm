Installing and Uninstalling Add-ons

You can use the Plug-in Manager to install and uninstall an add-on (.xsiaddon).

To install an add-on

  1. In the Plug-in Tree, right-click the User Root or first workgroup in the tree, and select Install .xsiaddon. The Select Add-on File dialog appears.

    To install the add-on in a different workgroup, select the Workgroups tab and move that workgroup to the top of the list. You can install add-ons only in the first workgroup.

  2. Locate the .xsiaddon file you want to install, and click OK.

NOTE:You can also install an add-on by dragging an .xsiaddon file to a Softimage viewport. This action installs the add-on in the User location or first workgroup depending on the value in the DefaultDestination tag of the add-on.

To uninstall an add-on

  1. In the Plug-in Tree, right-click an add-on and select Uninstall Addon. A confirmation dialog appears.
  2. Click Yes to uninstall.
