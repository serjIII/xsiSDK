Identifying Guide Objects in the Default Guides

If you want to create your custom rig based on Softimage's default proportional guides, you'll need to know the names of the objects to use as the guide objects.

The guide objects are the cubes you manipulate in the default rigs--either the red cubes for bone positions or yellow ones for volume deformer positions

For example, the MakeBelly command requires the names of the upper stomach and lower stomach guide markers to help define the stomach's volume. In the default rig, these are called UpperStomach and LowerStomach.
