Head and Neck

The MakeHead command corresponds to the options on the Head tab. It is used for both bipeds and quadrupeds.

To automatically create the head controls, the command requires guide objects that define the head, the neck, and the base of the neck. If you supply guides for the ears, the command creates the control objects.
