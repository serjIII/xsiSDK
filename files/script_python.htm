Python Scripting

This section answers some of the questions or issues that you might have when working with Python ActiveX Scripting.

Python ActiveX Scripting is a version of the Python scripting language that supports ActiveX. Softimage automatically installs the following for Windows and Linux operating systems.

  • Python v2.7.3
  • The pywin32 extensions v217

If you want to use a different version of Python in Softimage, you can set the location of the Python directory to use with the PYTHONPATH environment variable.
