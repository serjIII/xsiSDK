ERROR : Syntax error

'ERROR : Syntax error - [line 1]

Possible Cause

You may have forgotten to use parentheses or some other special character (such as an end of line character, or you may have used an illegal character. For example, if you try to use == as a comparison operator in VBScript.

Suggested Solution

Check the syntax of the line reported in the error message.
