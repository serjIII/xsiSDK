Chest and Spine

The MakeTorso command corresponds to the options on the Chest tab. It is used for both bipeds and quadrupeds.

To automatically create the torso controls, the command requires guide objects that define the spine, legs, and shoulders. From these guide objects, the command creates the chest, hip, and spine control objects.
