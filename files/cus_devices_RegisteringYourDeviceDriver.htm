Registering Your Device Driver in Softimage

Testing Your Device Drivers

When you are ready to test your device driver, you need to install your .xsidevice file, library file, any SPDL and preset files, and any extra support files in Softimage.

To install your device driver plug-in for testing

  1. Copy the .xsidevice file to one of the following locations so that Softimage can read it and load the device in the DeviceManager window the next time you start Softimage:

    • $factory/Data/Devices

    • $user/Data/Devices

    • $workgroup/Data/Devices

  2. Copy the library file to the $factory\Application\bin directory so that Softimage can find it the next time you start Softimage.
