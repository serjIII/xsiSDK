Numeric Controls

There are three kinds of numeric controls available:

  • Numeric Edit Box with Slider —standard Softimage numeric parameter control.

  • RGB Color Control —standard Softimage color parameter control without the alpha channel.

  • RGBA Color Control —standard Softimage color parameter control including the alpha channel.

The user can change the values that appear in these controls directly, but they can also be changed dynamically through Logic (see Dynamically Changing Custom Properties).
